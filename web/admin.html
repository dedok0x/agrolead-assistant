Как заполнять сценарии

					
1) Добавляйте кейсы по структуре:
- цель клиента
- обязательные уточнения (товар, класс, объём, регион, срок)
- критерий передачи в лид

2) Для каждого сценария задавайте:
- шаблон короткого ответа
- условия отказа/оффтопа
- обязательный финальный вопрос для квалификации.

3) Регулярно проверяйте:
- долю отказов по оффтопу
- среднюю длину ответа
- процент диалогов, где запрошен контакт.
