# MVP-сценарии

## Сценарий A — Запрос цены/наличия

1. Клиент отправляет запрос по товару.
2. Система определяет товар и ищет данные.
3. Клиент получает ответ по цене и доступности.

## Сценарий B — Уточнение параметров

1. Система задаёт уточняющие вопросы (объём, класс, срок).
2. Формирует квалифицированный интерес.

## Сценарий C — Создание заявки

1. На основе подтверждённого интереса создаётся заявка.
2. Менеджер получает заявку в админ-интерфейсе.

## Сценарий D — Просмотр обращений менеджером

1. Менеджер открывает список новых заявок.
2. Просматривает контекст диалога.
3. Меняет статус обращения.
